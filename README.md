Pong by Elena
------------------------------------------------------------------------------------------------------
Project Description:

This project is a two-player Pong game made with Python and the turtle module. Player A uses W and S, and Player B uses the up and down arrow keys. The goal is to bounce the ball past the other player and reach the winning score first.

How to Install and Run the Project

1.Make sure Python is installed on your computer.

2.Save the game code in a file like pong.py.

3.Open the file in a Python editor or IDE.

4.Run the file.

5.Use W and S for Player A, and the up and down arrow keys for Player B.

-----------------------------------------------------------------------------------------------
Features:

-Two-player gameplay.

=Score tracking for both players.

-Winner message and celebration when a player reaches the max score.

-Simple black-background arcade style.

--------------------------------------------------------------------------------------------
Controls:
-Player A: W to move up, S to move down.

-Player B: Up Arrow to move up, Down Arrow to move down.
